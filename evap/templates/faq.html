    {% trans "These are the Frequently Asked Questions (FAQs)" %}

    {% trans "General" %}

    {% trans "What is evaluation?" %}

    {% blocktrans %} It's the assessment of all the courses at the HPI. It's the goal to improve the teaching quality by giving the lecturers feedback for their courses. {% endblocktrans %}

    {% trans "What is EvaP?" %}

    {% blocktrans %} It's this new platform - replacing the infamous EvaJ. {% endblocktrans %}

    {% trans "Where are the data coming from?" %}

    {% blocktrans %} The different information, imported into the system, are provided by the Studienreferat. After that the lectures, their proxies or the student representatives correct any mistakes. If you just found another error please contact the responsible lecturer or the student representatives. {% endblocktrans %}

    {% trans "Questionnaires" %}

    {% blocktrans %} Explaining Questionnaires concept {% endblocktrans %}
    {% trans "How are questionnaires constructed?" %}

    {% blocktrans %} One questionnaire consists of different sub-questionnaires, e.g. about the contents or the teaching assistant. Each sub-questionnaire consists of one ore more questions. {% endblocktrans %}

    {% trans "What kinds of questionnaires are there?" %}

    {% blocktrans %} There are two kinds of questionnaires: one is about every person involved and one about an course itself. {% endblocktrans %}

    {% trans "What types of questions are there?" %}

    {% blocktrans %} There are two general types of questions. The first is called "Likert" and consists of five possible answers - from "strongly agree" to "strongly disagree". Additionally there is the option to give "no answer".
    With the second type it is possible to leave text comments, if the other questions miss an interesting point. {% endblocktrans %}

    {% trans "What types of courses are there?" %}

    {% blocktrans %} There are different types of courses, e.g. lectures, seminars, exercises and some more. {% endblocktrans %}

    {% trans "What kind of lecturer roles are there?" %}

    {% blocktrans %} Accordingly there are different kinds of lecturers, like lecturers, seminar or excesses supervisors. {% endblocktrans %}

    {% trans "How are questionnaires assigned to courses?" %}

    {% blocktrans %} The assignment is done by the student representatives. But the lecturers can add additional questionnaires. {% endblocktrans %}

    {% trans "Users" %}
    {% trans "How do I log into the system?" %}

    {% blocktrans %} If you are an HPI student or lecturer you can just use your usual credentials. If you are an external student or lecturer you can get an access token by Email. {% endblocktrans %}

    {% trans "How are users registered? Who is able to create new users?" %}

    New users are created by the Student Representatives. Ask them if there is the need for new users.

    {% trans "What are proxies?" %}

    {% blocktrans %} As professors do not have much time, it is possible for them to elect one or more proxies, e.g. a secretary or an employee of the chair. These people then can edit the courses as the original lecturer could, i.e. add questionnaires and setup the correct assistants and evaluation period. {% endblocktrans %}

    {% trans "Publication of results" %}
    {% trans "How are the results publicised?" %}

    {% blocktrans %} The results are viewable in the EvaP system, after the exams for the course are done. Later the student representatives will hang out a hard copy of the results. {% endblocktrans %}

    {% trans "What information are in the printed export?" %}

    {% blocktrans %} These prints contain the exact same information as shown in this system. The affected lecturers can also view the comments, which were given to them. {% endblocktrans %}

    {% trans "How is the overall grade calculated?" %}

    {% blocktrans %} The overall grade is the arithmetic average of all sub-grades. All personal questions together are equally weighted to the questions about the course itself. {% endblocktrans %}

    {% trans "How many answers are necessary to assure the representativeness?" %}

    {% blocktrans %} Currently at least five (5) answers are necessary (except "no answer") to assure the representativeness. If you do not agree with that write an Email or just come over to start a discussion with the student representatives. {% endblocktrans %}

    {% trans "Details for Lectures" %}
    {% trans "What data are necessary for a course and why?" %}

    {% blocktrans %} You can define all involved persons and their roles. So it is possible for the students to give specific feedback. Additionally you can adapt the evaluation period if needed. {% endblocktrans %}

    {% trans "How should I configure the of my course?" %}

    The evaluation must not influence the assessment of the student. So there are different criteria for the period:

    • It should end before the written exam or the first oral exam.
    • It should be at least one week long.
    • It should start after one of the last lectures and not right at the beginning.
    Of course there are courses for which it is not possible to fulfil all of the criteria. Please contact the student representatives if you are not sure.

    {% trans "When will I get which Email notice?" %}

    {% blocktrans %} At some point in the middle of the semester you get an Email with all your courses to edit them if needed. You will also get an Email after the results are available. {% endblocktrans %}

    {% trans "Which results do I see and what do they mean?" %}

    {% blocktrans %} You can see the average grade and standard deviation of all Likert questions. Additionally you can view the comments the students gave to you, the course or the other people involved in the course. {% endblocktrans %}

    {% trans "Details for Students" %}
    {% trans "How complete do I have to fill in the questionnaires?" %}

    {% blocktrans %} As complete as possible :). The results are only published if the representativeness is guaranteed. All Likert questions need to be answered (at least with "no answer"), but not one text comment question, to submit an questionnaire. {% endblocktrans %}

    {% trans "What happens to my comments?" %}

    {% blocktrans %} As we cannot assure the representativeness or anonymity here all comments are provided to all people involved in the course they were given. But the student representatives reserve the possibility to remove or change insults or other inappropriate texts. {% endblocktrans %}

    {% trans "How anonymous is my vote?" %}

    {% blocktrans %} The system guarantees that there is no connection between a vote and a user. It is just stored which user already submitted which questionnaire. Pay attention: We cannot assure anonymity regarding text comment questions as it is possible for the lecturer to guess from the contest or grammar of the text the student writing it. All text answers are given in an arbitrary order - so you should not refer to your own answers (like "see above" or "same as other teaching assistant"). {% endblocktrans %}
